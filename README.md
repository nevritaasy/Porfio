# **Porfio.**

## **🌐 Porfio: Part of Your Journey**

Porfio adalah aplikasi berbasis AI dan cloud yang memungkinkan pengguna mengunggah CV atau portfolio mereka dan kemudian sistem akan,

* Menganalisis skill dan pengalaman pengguna
* Melakukan kategorisasi bidang karir yang sesuai
* Memberikan rekomendasi pekerjaan dan saran kepada pengguna
